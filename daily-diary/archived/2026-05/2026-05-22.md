**ican2u — Tetapan Sistem Menu Visibility Fix**

📌 Focus  : Diagnose why "Tetapan Sistem" menu was not visible after creation
✅ Result : Root cause found and fixed — menu now visible

**Investigation**
1. Menu items existed in DB (IDs 160, 161) ✅
2. `app_function` record existed (ID 125) ✅
3. Menu rendered from **Meilisearch**, not MySQL — new DB records were never synced ❌
4. Administrator role (id:1) missing 4 permissions — `addIfCan('.view')` = menu hidden ❌

| Fix | What |
|-----|------|
| `php artisan scout:import App\Models\MenuMgmt\MenuItem` | Synced all 161 menu items to Meilisearch |
| DB insert `tbl_role_has_permissions` | Added permissions 255–258 to Administrator (id:1) |

⚠️ **Rule to Remember:** After inserting menu items to DB, always run `scout:import` for `MenuItem` — Meilisearch serves the menu, not MySQL directly.
