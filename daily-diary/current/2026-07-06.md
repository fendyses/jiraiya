### Key Insights & Learning
- Google's OpenAI-compatibility layer for Gemini is a genuine drop-in replacement for the OpenAI chat-completions shape (models list, chat, function calling, tool-result feeding) — no translation layer needed, unlike native Gemini SDK shapes.
- Gemini's `/models` endpoint has no `supported_features`-style metadata like Groq/OpenRouter do; had to default `supportsTools: true` for every remaining chat model after exclusion-filtering, verified correct via a live tool-call test.
