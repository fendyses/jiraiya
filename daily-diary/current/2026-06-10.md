**apps-back-end — Admin Student Status Toggle (LULUS ↔ GAGAL)**

📌 Focus  : Allow Administrator role to toggle student result status directly from the student listing table on program detail pages
✅ Result : Toggle button with SweetAlert2 confirmation popup implemented across all 4 program types

**What Was Built**

- New `POST` endpoint: `program/toggleStudentStatus/{id}/toggle` → `ProgramController@toggleStudentStatus`
  - Restricted to `Administrator` role (returns 403 otherwise)
  - Only `LULUS` and `GAGAL` statuses are toggleable (other statuses return 422)
  - Records `updated_by` on save

- `indexLatihan()` updated to pass `isAdmin` flag to the view

- Toggle button (🔄 `refresh-cw` icon) rendered to the **right** of the status badge — visible to admins only, only for LULUS/GAGAL rows

- SweetAlert2 confirmation popup on click:
  > *"Do you want to change status from LULUS to GAGAL?"*
  with **Yes** / **Cancel** — no change made on cancel
