### Key Insights & Learning
- A Flutter iOS archive and an App Store IPA are separate outcomes: a Personal Team can build and install a development-signed app on a registered device but cannot create an App Store distribution profile.
- Xcode's “iOS is not installed” destination error can occur even when the device SDK exists; the matching platform runtime must also be registered, and the developer disk image must be available for device operations.
- Developer Mode is mandatory for command-line Xcode installation on a physical iPhone.
- iOS launcher icons must be supplied at every required asset-catalog size and should not contain an alpha channel.
- API keys passed through Dart defines are embedded in the compiled client application, which is acceptable for this private testing phase but must be moved behind a backend before public distribution.
